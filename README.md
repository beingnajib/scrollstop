# ScrollStop

ScrollStop is an Android application built using Kotlin and Jetpack Compose.

## Features
- Built with Kotlin and modern Android architecture.
- Full Jetpack Compose UI implementation.
- Coroutines for asynchronous operations.
- Room for local database management.
- DataStore for preferences.

## Requirements
- Android SDK 35
- Minimum SDK 26

## Setup & Run
To run this project:
1. Open the project in Android Studio.
2. Build the project using Gradle.
3. Run on an emulator or physical Android device running API level 26 or higher.

## Architecture
The application is structured using standard Android modern development practices.
